# Results

This folder contains compact, Git-friendly experiment summaries reconstructed
from the completed test evaluation.

- `test_objective_summary.csv`: weighted overall metrics
- `test_objective_by_task.csv`: task-level metrics
- `grpo_pilot_vs_final_by_task.csv`: direct checkpoint judge comparison
- `selected_checkpoint.json`: final checkpoint decision

Large response-level files and raw checkpoints should remain outside GitHub.
